# [NodeJS](https://nodejs.org/) &middot; 

## Instruções básicas

### TypeORM
- Sequelize

### Banco de dados 
- sqlite3

### Instalar as dependências
- ```npm install``` ou ```yarn install```

### Criar arquivo do banco 
- npx sequelize-cli db:migrate

### Rodar API
- ```npm start``` ou ```yarn start```
